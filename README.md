# Startup Autopsy

A Claude Code skill that generates visual HTML presentations diagnosing startup failure risks. Input your industry and business model, get a graveyard-themed case-study deck with cross-industry failure patterns, real data from 89 YC startups (2017-2025), and survival strategies.

## How It Works

1. Pick your industry + business model
2. Choose a visual theme (default: Forensic Dark)
3. Get a 12-slide presentation with:
   - Pattern-matched failure analysis across industries
   - Real case studies with specific numbers
   - Survival playbook derived from what killed similar companies

## Presentation Engine

Visual output powered by [frontend-slides](https://github.com/zarazhangrui/frontend-slides) by [@zarazhangrui](https://x.com/zarazhangrui). Style presets, viewport fitting, and animation patterns all come from her skill.

## Data Source

Company data sourced from [startups.rip](https://startups.rip), enriched with Crunchbase, TechCrunch, and founder post-mortems.
