# frontend-slides

Presentation engine by [@zarazhangrui](https://github.com/zarazhangrui/frontend-slides)

This directory contains files from the [frontend-slides](https://github.com/zarazhangrui/frontend-slides) skill, which provides viewport-fitting CSS, animation patterns, and curated style presets for HTML presentations.

All credit for the presentation infrastructure goes to @zarazhangrui.
startup-autopsy uses these files to generate its visual output.
