# ui-ux-pro-max

Design data from [ui-ux-pro-max](https://github.com/anthropics/ui-ux-pro-max-skill) skill v2.0.1.

Files included:
- `styles.csv` — 50 design styles with CSS keywords, implementation checklists
- `colors.csv` — 21 color palettes by product type
- `typography.csv` — 50 font pairings with Google Fonts URLs
- `charts.csv` — 20 chart types with best-use guidance
- `ux-guidelines.csv` — UX best practices
- `ui-reasoning.csv` — UI decision reasoning patterns

Used by startup-autopsy for design intelligence when generating HTML presentations.
